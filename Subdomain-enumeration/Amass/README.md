# ONE-LINE COMMANDs

## Passive
```sh
amass enum -passive -d hackerone.com -o passive_h1.txt -rf 50resolvers.txt
```

## Active
```sh
amass enum -active -d hackerone -brute -o active_h1.txt -norecursive -noalts -rf 50resolvers.txt
```
