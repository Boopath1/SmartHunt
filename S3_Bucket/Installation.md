# Amazon Bucket S3 AWS

## Install awscli

```bash
sudo apt install awscli
```

## Configure awscli

```javascript
aws configure
AWSAccessKeyId=[ENTER HERE YOUR KEY]
AWSSecretKey=[ENTER HERE YOUR KEY]
```

## Set-up Config

```javascript
mousepad ~/.aws/config
```

### It should look similar to

```bash
[default]
region = us-west-2
aws_access_key_id = AKI************
aws_secret_access_key = Isx8***************
```
